Oi, se você está lendo esse texto muito provavelmente eu fiz algo que pode ser útil à você.

Primeiramente: se trata de um código em JavaScript feito para ser interpretado como com uma “extensão” do Google Chrome.
 A lógica por trás do código é simple: O navegador precisa acessa a página na qual contém o próximo dia disponível para a entrevista
 e então faz uma comparação com um data informada por você, se a data que ele encontrou estiver mais próxima que a data informada pelo usuário ele emitirá uma NOTIFICAÇÃO
 (lembre-se de habilitar as notificações do Chrome) informando a data encontrada. Todo o processo é reiniciado a cada x segundo (você decide! Recomendo 60s)


1 -> Você precisará de algumas ferramentas para te auxiliar a utilizar esse bot,
Um editor de texto avançado:
    Visual Studio Code: https://code.visualstudio.com/ ou NotePad++: https://notepad-plus-plus.org/downloads/
    Google Chrome e winrar.

2 -> Você precisa editar o arquivo "content.js" colocando os seus dados

3 -> Faça testes para ver se o bot está funcionando corretamente. 
    Como faço teste? Informe uma Data Marcada superior a data que atualmente está aparecendo no site, de modo que seja possível "forçar" a ativação do bot para verificar se o sistema de notificação está funcionando.
    A maior dor de cabeça que vai te acontecer é quando o bot está funcionando perfeitamente porém o seu PC não está notificando, verifique por isso.
