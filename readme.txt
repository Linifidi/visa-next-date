Oi, se você está lendo esse texto muito provavelmente eu fiz algo que pode ser útil à você.

Primeiramente: se trata de um código em JavaScript feito para ser interpretado como com uma “extensão” do Google Chrome.
 A lógica por trás do código é simple: O navegador precisa acessa a página na qual contém o próximo dia disponível para a entrevista
 e então faz uma comparação com um data informada por você, se a data que ele encontrou estiver mais próxima que a data informada pelo usuário ele emitirá uma NOTIFICAÇÃO
 (lembre-se de habilitar as notificações do Chrome) informando a data encontrada. Todo o processo é reiniciado a cada x segundo (você decide! Recomendo 60s)
